# Домашнее задание №2

## Цель
Написать свой "Hello, World!" модуль ядра с использованием параметров и callback-функций

## Инструкция по выполнению
- вывести фразу "Hello, World!" необычным способом
- в параметр **idx** передать индекс символа в массиве
- в параметр **ch_val** передать ASCII-код символа, который необходимо записать
- в параметре **my_str** будут отображаться символы, получаемые в результате

## Формат сдачи:
В проекте должены быть:
- **Makefile** содержит цели: format, check
- модуль выводит сообщение **"module_name: init"** в dmesg при insmod
- модуль выводит сообщение **"module_name: exit"** в dmesg при rmmod
- корректные ограничения и валидация параметров:
  - строковый параметр **my_str** должен быть read-only
  - **idx** - это индекс в массиве
  - **ch_val** - это ВИДИМЫЙ ASCII символ
- модуль собирается, загружается и выводит записать "Hello, World!"
